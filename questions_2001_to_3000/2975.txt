Problem Title: Maximum Square Area by Removing Fences
Brute Force Method: Try every combination of horizontal and vertical fence removals to form a square.
BF Complexity Visuals: Exhaustive Subset Removal
How to Draw BF: Draw a grid. Branch into 2^H x 2^V removal states. Impossible for large N.
Optimised Pattern: Hash Set Intersection (Difference Checking)
Opt Vis. Technique: Venn Diagram of Segment Lengths
Dry Run (Opt): Calculate all possible distances between any two horizontal fences and store in Set H. Do the same for vertical in Set V. The max common value in both sets is the side of the square.
How to Draw (Opt): Draw horizontal lines. Write all possible gap sizes (e.g., fence 1-3, 1-4, 2-4). Put them in a circle. Find the biggest number that appears in the vertical circle too.
Opt Complexity Visuals: Quadratic Set Population
How to Draw Opt Complexity: Draw two squares representing H^2 and V^2. T = $O(H^2 + V^2)$.
Mem Vis (BF): Storing all subset grid states.
Mem Vis (Opt): Two Hash Sets of sizes H^2 and V^2. Space $O(H^2 + V^2)$.
