Problem Title: Sum of Squares of Special Elements
Brute Force Method: Iterate through the array. For each element at index i, check if n % (i + 1) == 0. If it is, square it and add to sum. $O(N)$.
BF Complexity Visuals: Linear Conditional Scan
How to Draw BF: Draw the array. For every box, perform a division check. If it passes, draw a "Square" above it and move to the sum.
Optimised Pattern: $O(1)$ State Variables (Draw one sum and one i).
Opt Vis. Technique: Divisor Enumeration (O(√N))
Dry Run (Opt): Symmetric Divisor Sweep
How to Draw (Opt): Instead of checking all numbers up to N, find all divisors of N by iterating only up to √N. For each divisor d, both d and N/d are indices to check.
Opt Complexity Visuals: Draw a number line. Only pick specific boxes that are factors of the total length. Highlight how many boxes you skip.
How to Draw Opt Complexity: Square Root Time $O(√N)$
Mem Vis (BF): Draw a loop that stops halfway. Label it $O(√N)$ to show significant speedup for large arrays.
Mem Vis (Opt): Register Accumulator (Draw a single 64-bit integer cell for the sum).
