Problem Title: Binary Watch
Brute Force Method: Iteration Check all 720 possible times (12 hrs * 60 mins). Count set bits.
BF Complexity Visuals: 2D Grid
How to Draw BF: Draw a 12x60 grid. Shade squares where bit-count == turnedOn. Write $O(1)$ next to it (constant bounded size).
Optimised Pattern: Backtracking Iterate through the 10 LEDs, choosing to turn them on/off until `turnedOn` is met.
Opt Vis. Technique: Decision Tree
Dry Run (Opt): Draw root, branch left (LED off) and right (LED on). Stop branching when depth=10 or bits &gt; turnedOn.
How to Draw (Opt): Draw 10 circles (LEDs). Write '1' or '0' inside them. Keep a running tally of active LEDs.
Opt Complexity Visuals: Pruned Tree
How to Draw Opt Complexity: Draw a wide tree, but draw thick red "X" marks on branches that exceed the required bit count to show pruning. $O(10 C k)$.
Mem Vis (BF): Variables 2 integer counters for H and M.
Mem Vis (Opt): Call Stack Draw stacked blocks up to height 10 representing recursive states.
