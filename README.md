# LeetCode Visual Study Guide (Downloadable)

A premium, full-screen interactive study guide for all **3,873 LeetCode questions**, combined into a single downloadable HTML file (`master.html`) for full offline use.

## Features
- 🟡 Deep black background with glowing yellow UI theme
- 🔍 Live search across all questions (ID, title, pattern)
- 📐 MathJax rendering for all Big-O complexity notation: $O(N \log N)$, $O(N^2)$ etc.
- 📜 Continuous vertical feed — no pagination
- 🖥️ Fits the full screen width and height

## How to Run Locally
```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deployment
Live on **GitHub Pages** at: `https://<your-username>.github.io/<repo-name>/`
