ID: 1739
Problem Title: Building Boxes
Brute Force Method: Add boxes one by one, ensuring each is supported. Count total until it exceeds `n`.
BF Complexity Visuals: Layer-by-Layer Physical Simulation.
How to Draw BF: Draw a pyramid base. Add one box at a time to the edge. $O(\text{sqrt}(N)$).
Optimised Pattern: Mathematical Series (Tetrahedral Numbers) + Greedy.
Opt Vis. Technique: Staircase Growth Pattern.
Dry Run (Opt): Use `i*(i+1)*(i+2)/6` to find the largest complete tetrahedral pyramid. Then add leftover boxes in a triangular sequence.
How to Draw (Opt): Draw a 3D pyramid. Calculate its base. For leftovers, draw a new 2D triangle attached to the base.
Opt Complexity Visuals: Mathematical Convergence.
How to Draw Opt Complexity: Draw a number line. Jump to the nearest pyramid size, then step to the answer. $O(N^(1/3)$).
Mem Vis (BF): None, just counters.
Mem Vis (Opt): $O(1)$ variables for current height and base.
