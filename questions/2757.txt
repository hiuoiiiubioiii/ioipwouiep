Problem Title: Generate Circular Array Values
Brute Force Method: Simulate the rotation by actually slicing and concatenating the array or moving an index one-by-one in a while loop. $O(\text{Steps})$.

How to Draw (Opt): Calculate the new index directly using (current + steps % n + n) % n. This handles large positive and negative steps instantly in one operation.
