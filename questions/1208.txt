ID: 1208
Problem Title: Get Equal Substrings Within Budget
Brute Force Method: Generate every possible substring (start index i, end index j). For each, calculate the total absolute difference cost. Find the maximum length substring where the total cost is &lt;= maxCost.
BF Complexity Visuals: Triangle of Substrings
How to Draw BF: Draw a string. Draw a bracket under index 0. Then brackets under 0-1, 0-2... then 1-2, 1-3. The expanding brackets form a pyramid representing $O(N^2)$ or $O(N^3)$ if re-summing every time.
Optimised Pattern: Sliding Window (Variable Size)
Opt Vis. Technique: Expanding and Contracting Caterpillar
Dry Run (Opt): Imagine a caterpillar eating characters. As it eats the right character, it absorbs its "cost". If its belly gets too full (cost &gt; maxCost), it must poop out the left character (subtract left cost). Keep track of its maximum physical length.
How to Draw (Opt): Draw two parallel strings (s and t). Write the absolute differences between them in an array below. Draw 'L' and 'R' pointers starting at index 0. Move 'R' right, keeping a running sum. If sum &gt; maxCost, move 'L' right.
Opt Complexity Visuals: Two Independent Runners
How to Draw Opt Complexity: Draw an array. Draw an 'R' arrow moving continuously right. Draw an 'L' arrow occasionally moving right, but never passing 'R' and never moving left. Write $O(N)$ because both pointers only traverse the array once.
Mem Vis (BF): Draw hundreds of short, temporary string snippets generated in memory for every substring check.
Mem Vis (Opt): Draw exactly 4 static integer boxes: 'left_pointer', 'right_pointer', 'current_cost', and 'max_length'. Representing $O(1)$ auxiliary space.
