Problem Title: Count of Integers
Brute Force Method: Iterate from `num1` to `num2`. Convert each number to a string, sum its characters, and check if it falls between `min_sum` and `max_sum`. TLEs massively on 10^22.
BF Complexity Visuals: Massive Number Line Scanning
How to Draw BF: Draw a giant number line. Draw a magnifying glass iterating over every single tick mark. Label it $O(N)$ where N is the massive numeric difference.
Optimised Pattern: Trillions of temporary string allocations for digit extraction.
Opt Vis. Technique: Digit DP + Memoization `solve(num2) - solve(num1 - 1)`
